# Library-Management-System-cpp
Library Management System (C++) 
This is a console-based Library Management System developed using C++ and Object-Oriented Programming (OOP) concepts. 
The program helps users manage books in a small library by allowing them to:
Add new books,
View all available books,
Issue books,
Return books.
This project was created for learning and practicing C++ programming and OOP concepts.
## Features
It has concepts of 
Encapsulation, 
Objects, 
Classes, 
Private & Public specifiers, 
Loops and Conditional statements.
